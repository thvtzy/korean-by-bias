# Korean by Bias

A vocabulary tracker built for K-pop fans learning Korean. Save words you hear in songs, lives, and variety shows — then review them with spaced-repetition flashcards.

## Status

In active development. Built as part of a 3-month learning sprint.

## Roadmap

- [x] Landing page (week 1)
- [ ] Add/list vocab with localStorage (week 2-3)
- [ ] Flashcard review mode (week 4-5)
- [ ] Cloud sync via Supabase + auth (week 6)
- [ ] Auto-translate (week 7+)
- [ ] Multi-language support (Japanese, Mandarin)

## Stack

- Vanilla HTML / CSS / JavaScript (no framework)
- Vercel for hosting
- Supabase for auth + database (later phases)

## Local development

Just open `index.html` in a browser. No build step.
